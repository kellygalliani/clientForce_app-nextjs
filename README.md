## 4. Instalação e uso

### 4.1 Instalação:

git clone git@github.com:kellygalliani/clientForce_app-nextjs.git
<br>
cd clientForce_app-nextjs.git
<br>
npm install
<br>
npm run start:dev
<br>
optional: include .env in your .gitignore
